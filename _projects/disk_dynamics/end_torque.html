from scipy.interpolate import interp1d
from scipy.stats import sem 

def get_torque(base, n_max=50, suffix=''):
    a_std=np.linspace(1.01, 1.99, 1000)
    tauc=np.zeros(n_max)
    for ii in range(n_max):
        dat2=np.genfromtxt(base+'_{0}'.format(ii+1)+suffix)
        order=np.argsort(dat2[:,0])
        dat2=dat2[order]
        dat2[:,1]=np.cumsum(dat2[:,1])
        tauc[ii]=(interp1d(dat2[:,0], abs(dat2[:,1]))(a_std[-1]))

    return np.mean(tauc),sem(tauc)

Timescale to change angular momentum of a test orbit ($t_j=j/\tau$ in units of the orbital time) as a function of its semi-major axis and its orientation relative to the eccentric nuclear disk ($\bar{\omega}$). Each data point corresponds an average over 50 realizations of an eccentric nuclear disk with 1000 particles. All orbits have an eccentricity of 0.7.

We have normalized $t_j$ by the orbital time, which decreases with the semi-major axis. However, even without this normalization, $t_j$ is still much longer for smaller smas.
